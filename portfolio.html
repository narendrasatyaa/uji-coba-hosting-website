Simple Donation Management System with PHP & MySQL

This website is a simple donation management system built using PHP Native and MySQL as a database. This application allows users to manage donation data for underprivileged children, including features to add, edit, delete, and view a list of donations that have been received. With a simple and functional design, this system is suitable for use in social projects or as a learning tool in developing web-based applications using PHP Native and MySQL.
